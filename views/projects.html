Jeremy G Holland Blog

This is MEAN stack blogging app which currently hosts my baseball related blog posts. It allows for people to view current posts and comments related to individual posts, while also allowing for registered users to add posts and comments. This blog uses passport for verification and uses Materialize.css for its styling.

MarvelAPI

This is a search app using Marvel Comics's RESTful API to pull all events shared between two comic book characters. It uses jQuery for most of its DOM manipulation and as well for AJAX calls.

KittyWars

A trading game in which the player will travel and trade in kitten based wares, including catnip and scratching posts, to accumulate more Cheezeburgers. This MVC game utilizes AngularJS to provide javascript functionality and AngularFire as a backend database and authentication services.
